JendoubaLife Tourism API

Welcome to the JendoubaLife Tourism API. This API provides information about tourist places, events, reviews, and more in Jendouba, Tunisia.

API Routes

  • Users: /api/users
  • Places: /api/places
  • Events: /api/events
  • Reviews: /api/reviews
  • Reservations: /api/reservations
  • Promotions: /api/promotions
  • Messagerie: /api/messagerie

Reservation Endpoints

  • GET /api/reservations - Get all reservations
  • GET /api/reservations/:id - Get a specific reservation
  • POST /api/reservations - Create a new reservation
  • PUT /api/reservations/:id - Update a reservation
  • DELETE /api/reservations/:id - Delete a reservation
  • GET /api/reservations/availability - Check availability
